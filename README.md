# dart-cli-crypto-triple-des-encrypt-bcrypt-encoded

## Description
Comparing passwords with one-way hashes.
Encryptes password before encoding to ensure
unique and stronger hashes.

## Tech stack
- dart
  - dargon2
  - steel_crypt
  - dart_des

## Docker stack
- google/dart

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
